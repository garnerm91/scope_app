# Scope App

A Python desktop oscilloscope viewer built with Dear PyGUI. The app connects to VISA-capable instruments when available and can fall back to simulated waveform data.

## Features

- Live waveform plotting
- Multiple channel selection
- FFT and jitter analysis views
- Manual VISA command send/query
- CSV export for PulseView import

## Requirements

Install the Python dependencies:

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```
